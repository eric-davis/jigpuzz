# JigPuzz

[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](LICENSE)

A browser-based jigsaw puzzle game with zero dependencies. Just open `index.html` and start puzzling.

**Live demo:** https://esdavis.dev/jigpuzz

**Test environment:** https://test.esdavis.dev/jigpuzz — latest `main` build.

## Features

- **Image upload** — use any image as your puzzle
- **Piece count** — 12 to 400+ pieces, or Auto mode that picks a good count for your screen
- **Piece rotation** — optional, togglable; rotate with scroll wheel, right-click, or R key
- **Drag & drop** — pieces snap together automatically when placed close enough
- **Peek mode** — hold P or the peek button to see the reference image
- **Win detection** — confetti, animation, and elapsed time on completion
- **Sound effects** — mutable
- **Desktop & mobile** — pointer and touch events supported
- **URL persistence** — seed, piece count, and rotation toggle saved in the URL hash

## Usage

Open `index.html` in any modern browser. No server, no build step, no dependencies.
